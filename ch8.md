# Introduction to Python Modules

## 1. Concept of Modules

-   **Module**: A file containing Python definitions, statements, and functions. It helps in organizing code and reusing functions.

-   **Importing a Module**: Allows you to use the functions defined inside the module.
    -   **Syntax 1**: `import math` (Access functions as `math.sqrt()`)
    -   **Syntax 2**: `from math import sqrt` (Access function directly as `sqrt()`)

## 2. Common Modules

-   **`math`**: Provides mathematical functions (e.g., `sqrt()`, `pow()`, `pi`).
-   **`random`**: Provides functions for generating random numbers (e.g., `randint()`).
