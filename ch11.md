List Manipulation

### 1. List Basics

*   **Ordered Sequence**: Elements maintain their insertion order.
*   **Mutable**: Lists can be changed in place after creation.
*   **Heterogeneous**: Can hold items of different data types.

### 2. Accessing & Slicing

*   Same indexing and slicing rules as strings.

### 3. Key Methods

| Method           | Description                                                        |
| :--------------- | :----------------------------------------------------------------- |
| `L.append(item)` | Adds item to the end of the list.                                  |
| `L.insert(i, item)` | Inserts item at index `i`.                                         |
| `L.extend(L2)`   | Appends all elements of `L2` to `L`.                               |
| `L.remove(item)` | Removes the first occurrence of `item`.                             |
| `L.pop(i)`       | Removes and returns the element at index `i` (or the last element if `i` is omitted). |
| `L.sort()`       | Sorts the list in place.                                           |
| `L.index(item)`  | Returns the index of the first `item`.                             |
