# Getting Started & Python Fundamentals

## Key Characteristics

-   **Interpreted Language**: Python code is executed line-by-line by an interpreter.
-   **High-Level**: Closer to human language.
-   **Indentation**: Python uses indentation (whitespace) to define code blocks (e.g., after `if` or `for`). This is mandatory.

## Data Types

| Type    | Description                               | Examples                 |
| :------ | :---------------------------------------- | :----------------------- |
| `int`   | Integers (whole numbers).                 | `10`, `-5`, `0`          |
| `float` | Numbers with a decimal point.             | `10.5`, `-2.0`, `3.14`   |
| `str`   | Sequences of characters, enclosed in quotes. | `"Hello"`, `'Python'`    |
| `bool`  | Boolean values: `True` or `False`.        | `True`, `False`          |

## Operators

| Type       | Examples                                                  | Function                             |
| :--------- | :-------------------------------------------------------- | :----------------------------------- |
| Arithmetic | `+`, `-`, `*`, `/` (division), `//` (floor division), `%` (modulo - remainder), `**` (exponent) | Performs mathematical calculations.  |
| Relational | `==` (equal to), `!=` (not equal to), `>`, `<`, `>=`, `<=` | Compares two values; returns `True` or `False`. |
| Logical    | `and`, `or`, `not`                                        | Combines boolean expressions.        |
