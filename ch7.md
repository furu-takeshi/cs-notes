# Data Handling

## Input and Output

-   **Input Function**: The `input()` function is used to take data from the user. It always returns data as a string.
    -   **Syntax**: `variable = input("Prompt message: ")`

-   **Type Casting**: Converting data from one type to another (e.g., string to integer).
    -   **Syntax**: `age = int(input("Enter your age: "))`

-   **Output Function**: The `print()` function displays output to the console.
    -   Can print multiple values separated by commas (default separator is a space).

## Variables and Mutability

-   **Variable**: A named location in memory that stores data.
-   **Mutability**: Determines if the value of a data object can be changed after it is created.
    -   **Mutable**: Lists, Dictionaries (can be modified in place).
    -   **Immutable**: Integers, Floats, Strings, Tuples (a new object is created when their value changes).
